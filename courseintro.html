Introduction to the Course

Web App Foundations (ITMD-541)

This is Web Application Foundation Course ITMD-541, will help us to gain proper knowledge of HTML, CSS and JavaScript. This will help to enrich skills in term of Web development. This is Lab1 Assignment. In this course for HTML we will learn all the tags from start to sentimental tag. For CSS, We will be taught about all the properties and where to apply for creativity. In JavaScript, we will learn from basic to advance concepts level in development like AJAX, promises and closure etc.
